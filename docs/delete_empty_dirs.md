# delete_empty_dirs.ksh

指定したディレクトリ配下の空フォルダーを再帰的に削除するスクリプト。
削除後に空になった親フォルダーも繰り返し処理して確実に除去する。

## 使用方法

```sh
./delete_empty_dirs.ksh <対象ディレクトリ>
```

### 引数

| 引数 | 必須 | 説明 |
|---|---|---|
| `<対象ディレクトリ>` | 必須 | 空フォルダーを削除する親ディレクトリ |

### 実行例

```sh
./delete_empty_dirs.ksh /data/work
```

## 動作仕様

1. 対象ディレクトリ配下の空フォルダーを `find` で検出して削除する
2. 削除後に空になった親フォルダーも削除対象になるため、空フォルダーがなくなるまで繰り返す
3. 対象ディレクトリ自体は削除しない（`-mindepth 1` による）

## 出力メッセージ

```
削除: /data/work/2024/01/empty_dir
削除: /data/work/2024/01
完了: 2 件の空フォルダーを削除しました
```

## エラー処理

| エラー | 動作 |
|---|---|
| 引数の数が不正 | エラーメッセージを表示して終了（exit 1） |
| 対象ディレクトリが存在しない | エラーメッセージを表示して終了（exit 1） |

## 依存コマンド

| コマンド | 用途 |
|---|---|
| `find` | 空ディレクトリの検出 |
| `rmdir` | 空ディレクトリの削除 |
| `wc -l` | 件数集計 |
