# list_folder_structure.ksh

指定フォルダー配下のフォルダー構造を再帰的に調査し、フルパスのリストとしてファイルに保存するスクリプト。

## 使用方法

```sh
./list_folder_structure.ksh <対象フォルダー> [出力ファイル]
```

### 引数

| 引数 | 必須 | 説明 |
|---|---|---|
| `<対象フォルダー>` | 必須 | 構造を調査するフォルダー |
| `[出力ファイル]` | 省略可 | リストの出力先パス（省略時は自動命名） |

出力ファイルを省略した場合、実行ディレクトリに `folder_list_YYYYMMDD_HHMMSS.txt` を生成する。

### 実行例

```sh
# 出力ファイルを自動命名
./list_folder_structure.ksh /data/A

# 出力ファイルを指定
./list_folder_structure.ksh /data/A /tmp/folder_list.txt
```

## 動作仕様

- 対象フォルダー配下を `find` で**再帰的**に走査する
- ディレクトリ（`-type d`）のみ対象。ファイル・シンボリックリンクは含まない
- 対象フォルダー自身もリストに含まれる
- パスはソートしてフルパスのフラットリストとして出力する

## 出力例

```
/data/A
/data/A/sub1
/data/A/sub1/sub1-1
/data/A/sub2
```

## エラー処理

| エラー | 動作 |
|---|---|
| 引数の数が不正 | エラーメッセージを表示して終了（exit 1） |
| 対象フォルダーが存在しない | エラーメッセージを表示して終了（exit 1） |

## 依存コマンド

| コマンド | 用途 |
|---|---|
| `find` | 再帰的なフォルダー走査 |
| `sort` | パスのソート |
| `wc -l` | 件数集計 |
