# compare_folders.ksh

A フォルダーと B フォルダーのファイルをタイムスタンプとサイズで再帰的に比較し、差異をレポートファイルに出力するスクリプト。

## 使用方法

```sh
./compare_folders.ksh <Aフォルダ> <Bフォルダ> [出力ファイル]
```

### 引数

| 引数 | 必須 | 説明 |
|---|---|---|
| `<Aフォルダ>` | 必須 | 比較元フォルダー |
| `<Bフォルダ>` | 必須 | 比較先フォルダー |
| `[出力ファイル]` | 省略可 | レポートの出力先パス（省略時は自動命名） |

出力ファイルを省略した場合、実行ディレクトリに `diff_report_YYYYMMDD_HHMMSS.txt` を生成する。

### 実行例

```sh
# 出力ファイルを自動命名
./compare_folders.ksh /data/A /data/B

# 出力ファイルを指定
./compare_folders.ksh /data/A /data/B /tmp/report.txt
```

## 動作仕様

### 比較方法

`stat` コマンドで取得した以下の属性を比較する。ファイルの中身は読まない。

| 属性 | 説明 |
|---|---|
| タイムスタンプ | ファイルの最終更新時刻（Unix 秒） |
| サイズ | ファイルのバイト数 |

### 走査範囲

- A・B フォルダー配下を `find` で**再帰的**に走査する
- 通常ファイル（`-type f`）のみ対象。シンボリックリンク・ディレクトリは除外

### 差異の分類

| 分類 | 説明 |
|---|---|
| A のみ存在 | A にあって B に存在しないファイル |
| B のみ存在 | B にあって A に存在しないファイル |
| `[タイムスタンプ＋サイズ]` | 両方の属性が異なる |
| `[タイムスタンプ]` | サイズは同じ、タイムスタンプのみ異なる |
| `[サイズ]` | タイムスタンプは同じ、サイズのみ異なる |

### stat コマンドの互換性

起動時に `stat` の形式を自動判定する。

| 環境 | 使用するオプション |
|---|---|
| GNU stat（Linux） | `stat -c '%Y %s'` |
| BSD stat（macOS / Solaris） | `stat -f '%m %z'` |

## 出力レポート形式

レポートは標準出力にも表示され、同内容がファイルに保存される。

```
==================================================
ファイル差分レポート
実行日時  : 2026-06-05 12:00:00
Aフォルダー: /data/A
Bフォルダー: /data/B
==================================================

[Aのみ存在: 1件]
  logs/error.log

[Bのみ存在: 1件]
  archive.zip

[差異あり: 2件]
  [タイムスタンプ＋サイズ] data/report.csv
    A: 2024-01-15 09:00:00  1024 bytes
    B: 2024-01-20 11:30:00  2048 bytes
  [サイズ] config/settings.json
    A: 2024-03-01 08:00:00  512 bytes
    B: 2024-03-01 08:00:00  640 bytes

==================================================
集計: Aのみ 1件 / Bのみ 1件 / 差異あり 2件 / 合計 3件
==================================================
```

## エラー処理

| エラー | 動作 |
|---|---|
| 引数の数が不正 | エラーメッセージを表示して終了（exit 1） |
| A/B フォルダーが存在しない | エラーメッセージを表示して終了（exit 1） |
| `stat` コマンドが使用不可 | エラーメッセージを表示して終了（exit 1） |

## 依存コマンド

| コマンド | 用途 |
|---|---|
| `find` | 再帰的なファイル走査 |
| `stat` | タイムスタンプ・サイズ取得 |
| `date` | タイムスタンプの整形表示 |
| `mktemp` | 一時ファイルの作成 |
| `wc -l` | 件数集計 |
| `grep` | 差異件数のカウント |
