# copy_date_files.ksh

日付フォルダー内のファイルを B フォルダーへコピーするスクリプト。
同名ファイルが B に存在する場合は、上書き前に C フォルダーへバックアップする。

## 使用方法

```sh
./copy_date_files.ksh <ソースディレクトリ> <Bフォルダパス> <Cフォルダパス>
```

### 引数

| 引数 | 説明 |
|---|---|
| `<ソースディレクトリ>` | 日付フォルダー（YYYYMMDD）が格納された親ディレクトリ |
| `<Bフォルダパス>` | コピー先フォルダー |
| `<Cフォルダパス>` | 上書き前のバックアップ先フォルダー |

### 実行例

```sh
./copy_date_files.ksh /data/source /data/B /data/C
```

## 動作仕様

### 対象フォルダー

- ソースディレクトリ直下の `YYYYMMDD` 形式（8桁数字）のフォルダーを対象とする
- 対象範囲: `20201001` 〜 実行日当日
- 各日付フォルダーの**直下ファイルのみ**を処理する（サブディレクトリは無視）

### コピー処理

1. 日付フォルダーを昇順に走査する
2. B フォルダーに同名ファイルが**存在しない**場合 → そのままコピー
3. B フォルダーに同名ファイルが**存在する**場合 → C フォルダーへバックアップしてから上書き

### タイムスタンプの保持

`cp -p` を使用するため、コピー先のファイルは元ファイルのタイムスタンプ・パーミッションを保持する。

### バックアップファイルの命名規則

上書きが発生した際、既存ファイルを C フォルダーへ保存する。ファイル名には**新しいファイルが属する日付フォルダー名**を付加する。

| 元のファイル名 | C に保存されるファイル名 |
|---|---|
| `report.csv` | `report_20201015.csv` |
| `report` | `report_20201015` |
| `.hidden` | `.hidden_20201015` |

### フォルダーの自動作成

B・C フォルダーが存在しない場合は `mkdir -p` で自動作成する。

## 出力メッセージ

```
バックアップ: report.csv → C/report_20201015.csv
コピー: 20201015/report.csv → B/
...
完了: コピー 10 件, バックアップ 3 件, 失敗 0 件
```

| 項目 | 意味 |
|---|---|
| コピー | B フォルダーへコピーした件数 |
| バックアップ | C フォルダーへバックアップした件数 |
| 失敗 | コピーまたはバックアップに失敗した件数 |

## エラー処理

| エラー | 動作 |
|---|---|
| 引数の数が不正 | エラーメッセージを表示して終了（exit 1） |
| ソースディレクトリが存在しない | エラーメッセージを表示して終了（exit 1） |
| B/C フォルダーの作成失敗 | エラーメッセージを表示して終了（exit 1） |
| バックアップ失敗 | 警告を表示し、そのファイルのコピーをスキップして続行 |
| コピー失敗 | 警告を表示して続行 |

## 依存コマンド

| コマンド | 用途 |
|---|---|
| `cp -p` | タイムスタンプ保持コピー |
| `basename` | パスからファイル名を取得 |
| `mkdir -p` | フォルダーの再帰作成 |
| `date` | 今日の日付取得 |
