# compare_folders_content.ksh

A フォルダーと B フォルダーのファイル内容（バイト列）を再帰的に比較し、差異をレポートファイルに出力するスクリプト。
タイムスタンプ・パーミッションは比較対象外。

## 使用方法

```sh
./compare_folders_content.ksh <Aフォルダ> <Bフォルダ> [出力ファイル]
```

### 引数

| 引数 | 必須 | 説明 |
|---|---|---|
| `<Aフォルダ>` | 必須 | 比較元フォルダー |
| `<Bフォルダ>` | 必須 | 比較先フォルダー |
| `[出力ファイル]` | 省略可 | レポートの出力先パス（省略時は自動命名） |

出力ファイルを省略した場合、実行ディレクトリに `content_diff_report_YYYYMMDD_HHMMSS.txt` を生成する。

### 実行例

```sh
# 出力ファイルを自動命名
./compare_folders_content.ksh /data/A /data/B

# 出力ファイルを指定
./compare_folders_content.ksh /data/A /data/B /tmp/report.txt
```

## 動作仕様

### 比較方法

`cmp -s` コマンドでファイルのバイト列を直接比較する。

- 終了コード `0`：内容が同一
- 終了コード `1`：内容が異なる

`cmp -s` は最初の差異を検出した時点で処理を打ち切るため、大きなファイルでも効率的に動作する。

### 走査範囲

- A・B フォルダー配下を `find` で**再帰的**に走査する
- 通常ファイル（`-type f`）のみ対象。シンボリックリンク・ディレクトリは除外

### 差異の分類

| 分類 | 説明 |
|---|---|
| A のみ存在 | A にあって B に存在しないファイル |
| B のみ存在 | B にあって A に存在しないファイル |
| 内容が異なる | 両方に存在するが、バイト列が一致しないファイル |

## 出力レポート形式

レポートは標準出力にも表示され、同内容がファイルに保存される。

```
==================================================
ファイル内容差分レポート
実行日時  : 2026-06-05 12:00:00
Aフォルダー: /data/A
Bフォルダー: /data/B
==================================================

[Aのみ存在: 1件]
  logs/error.log

[Bのみ存在: 1件]
  archive.zip

[内容が異なる: 2件]
  data/report.csv
  config/settings.json

==================================================
集計: Aのみ 1件 / Bのみ 1件 / 内容差異 2件 / 合計 3件
==================================================
```

## エラー処理

| エラー | 動作 |
|---|---|
| 引数の数が不正 | エラーメッセージを表示して終了（exit 1） |
| A/B フォルダーが存在しない | エラーメッセージを表示して終了（exit 1） |
| `cmp` コマンドが見つからない | エラーメッセージを表示して終了（exit 1） |

## 依存コマンド

| コマンド | 用途 |
|---|---|
| `find` | 再帰的なファイル走査 |
| `cmp -s` | ファイル内容の比較 |
| `mktemp` | 一時ファイルの作成 |
| `wc -l` | 件数集計 |
| `date` | 実行日時の取得 |

## compare_folders.ksh との使い分け

| | compare_folders.ksh | compare_folders_content.ksh |
|---|---|---|
| 比較方法 | タイムスタンプ＋サイズ | バイト列（内容） |
| 速度 | 高速（ファイルを読まない） | ファイルサイズに依存 |
| 検出できるケース | タイムスタンプやサイズが異なるファイル | 内容が実際に異なるファイル |
| 検出できないケース | 内容は同じだが属性が異なるファイル | ―（バイト列で確実に判定） |

**推奨フロー**: まず `compare_folders.ksh` で高速に候補を絞り、必要に応じて `compare_folders_content.ksh` で内容を確認する。
